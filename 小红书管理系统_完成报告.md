### API 接口

```python
# 获取所有帖子
GET /api/posts

# 获取单个帖子
GET /api/posts/<filename>

# 创建新帖子
POST /api/posts
{
  "title": "标题",
  "tags": ["标签 1", "标签 2"],
  "content": "内容",
  "status": "published"
}

# 获取每日日志
GET /api/logs

# 生成每日内容
POST /api/generate-daily
```
