# 📕 小红书内容管理系统

## 🎯 功能概述

本系统提供完整的小红书内容创作、管理和发布功能：

1. **内容管理面板** (5010 端口) - 可视化管理所有帖子
2. **每日自动生成** - 根据系统变化自动创作 1-2 篇帖子
3. **帖子预览编辑** - 支持实时预览和编辑
4. **每日日志归档** - 自动记录系统变化

## 🎨 管理面板功能

### 5010 端口页面功能

1. **数据统计**
   - 总帖子数
   - 已发布数量
   - 草稿数量
   - 每日日志数量

2. **帖子管理**
   - 卡片式展示所有帖子
   - 支持预览、编辑
   - 状态标记 (草稿/已发布/计划发布)
   - 标签管理

3. **每日日志**
   - 时间线展示
   - 系统变化记录
   - 内容创作历史

4. **快速创建**
   - 点击右下角 + 按钮
   - 填写标题、内容、标签
   - 选择发布状态

## 🔄 自动化流程

### 每日自动执行 (08:00)

```
1. 获取系统状态
   ↓
2. 生成每日系统日志
   ↓
3. 生成 V1 帖子 (系统进展)
   ↓
4. 生成 V2 帖子 (交易心得)
   ↓
5. 保存到对应目录
   ↓
6. 管理面板自动更新
```

### 内容创作策略

**频率**: 每日 1-2 篇  
**时间**: 早上 8 点 (A 股开盘前)  
**类型**: 
- V1: 系统进展、功能更新
- V2: 交易心得、干货分享

**标签策略**:
- 固定标签：#量化交易 #AI #Q 脑
- 轮换标签：#技术分享 #投资理财 #副业 #个人成长

## 🛠️ 高级功能

### API 接口

```bash
# 获取所有帖子
GET http://localhost:5010/api/posts

# 获取单个帖子
GET http://localhost:5010/api/posts/<filename>

# 创建新帖子
POST http://localhost:5010/api/posts
{
  "title": "帖子标题",
  "tags": ["标签 1", "标签 2"],
  "content": "帖子内容",
  "status": "published"
}

# 获取每日日志
GET http://localhost:5010/api/logs

# 生成每日内容
POST http://localhost:5010/api/generate-daily
```

### 自定义模板

编辑 `daily_content_generator.py` 中的：
- `generate_daily_post_v1()` - V1 模板
- `generate_daily_post_v2()` - V2 模板

## 💡 常见问题

### Q: 如何修改自动生成的内容？
A: 编辑 `daily_content_generator.py` 中的模板函数

### Q: 可以手动创建帖子吗？
A: 可以！访问 5010 端口，点击右下角 + 按钮

### Q: 如何查看已发布的帖子？
A: 管理面板会自动显示所有帖子，包括状态标记

### Q: 定时任务不执行怎么办？
A: 检查 cron 服务状态：`sudo systemctl status cron`
